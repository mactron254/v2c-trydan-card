# Release notes — v0.4.1

Fecha: 2026-07-13

## Mejoras

- Editor visual completamente traducido en diez idiomas.
- Densidad XXL claramente mayor que estándar.
- Layouts centrado, dividido, en línea y automático con composición real.
- Selector de color personalizado con paleta y HEX sincronizado.
- Métricas y fuentes mediante chips; orden mediante botones; presets mediante tokens editables.
- Escala y radio con sliders y actualización inmediata.
- Estado “Sin vehículo” más cerca del cargador.
- 14 capturas reproducibles y guía visual completa.

## Compatibilidad

No añade ni elimina claves YAML. Configuraciones 0.4.0 siguen válidas.
