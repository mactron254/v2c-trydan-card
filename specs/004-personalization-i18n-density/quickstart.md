# Quickstart validation

1. Usar pnpm 11.5.1: `corepack pnpm typecheck`, `corepack pnpm test`, `corepack pnpm build`, `corepack pnpm smoke`.
2. Cargar demo con cada `display_mode` a 280, 320, 359, 360, 519, 520 y 768 px; confirmar contrato.
3. Probar diez idiomas: auto + explícito, estados V2C, advertencias, editor y modos de carga.
4. Probar discovery con dispositivo común, override válido, inválido, ambiguo y limpiado; nunca llamada servicio inválida.
5. Confirmar foco, teclado, zoom 200 %, tema claro/oscuro y movimiento reducido.
